    Imager Hover Effects & Hyperlinks

    The topic I decided to research is called Hover Effect. I have inserted 3 pictures with a hover effect on them but they also act like hyperlinks that direct the user to three different videos on my youtube channel. Image hover effects refer to the visual changes that occur when a user moves their cursor over an image on a website. These effects can add visual interest and interactivity to a website, and they can be achieved using a variety of techniques.

    Some common image hover effects include changing the opacity or saturation of the image, adding a border or a shadow, or displaying additional information or a different image.

    Image hover effects can help to make a website more engaging and interactive, and they can be used to add emphasis or to guide a user's attention to a particular image. I have used Image Opacity and Image Transparency for my example. “Opacity is the opposite of transparency, allowing no light to pass through. You can create transparent images in CSS with the opacity property”(corelangs) 1 is fully opaque and 0 is fully transparent. The lower the value of opacity, the more transparent the element will be.

    Image hover effects can be achieved using a variety of techniques, including changing the opacity or saturation of the image, adding a border or a shadow, or displaying additional information or a different image. There are three steps I followed and they are : “Selecting the element you want to apply a hover effect to, Switching the states menu to the hover state and adjusting styling, Switching back to the non state and adding a transition” (webflow)

    My project includes a few elements which include opacity, hover effect and a hyperlink. “A hyperlink is an element in an HTML document that links to either another portion of the document or to another document altogether.” (techopedia)

    These effects can be created using CSS, which is a style sheet language that is used to define the look and formatting of a document written in a markup language. By using CSS to create image hover effects, web designers and developers can have more control over the appearance and behavior of the images on their website.
